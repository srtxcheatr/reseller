How to Deploy on Your Website

1
Get your API key

Generate a key above. Copy it and keep it secure.

2
Make a request to the endpoint

Send a GET request with your API key as a query parameter.

GET https://srtxcheats.web.app/api/keys?apikey=YOUR_KEY
3
Parse the JSON response

The API returns all your purchased keys in JSON format.

{
        "success": true,
        "uid": "user_uid",
        "keys": [
          { "product": "DRIP 1 DAY", "key": "SRTX-XXXX-XXXX", "date": "Jun 27, 2026" }
        ]
        }
4
Display on your website

Use JavaScript to fetch and display keys to your users.

fetch('https://srtxcheats.web.app/api/keys?apikey=YOUR_KEY')
        .then(r => r.json())
        .then(data => {
          data.keys.forEach(k => {
            // render key on your page
          });
        });
